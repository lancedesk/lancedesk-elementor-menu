# Contributing to LanceDesk Elementor Menu

Thanks for your interest in improving LanceDesk Elementor Menu.

## Development Setup

1. Clone the repository.
2. Place the plugin in your local WordPress plugins directory.
3. Ensure Elementor is installed and active.
4. Activate the plugin and test changes in both Elementor editor and frontend.

## Contribution Guidelines

- Keep changes focused and small where possible.
- Follow WordPress coding standards and secure coding practices.
- Preserve backward compatibility unless a breaking change is intentional and documented.
- Avoid committing secrets, credentials, or local environment files.

## Before Submitting

- Test submenu behavior on desktop, tablet, and mobile breakpoints.
- Verify behavior in Elementor editor and frontend.
- Confirm there are no JavaScript console errors.
- Update docs (`README.md`, `readme.txt`, changelog) when behavior changes.

## Pull Request Checklist

- Clear title and summary of the change.
- Why the change is needed.
- How it was tested.
- Screenshots or short clips for UI/UX changes.

## Bug Reports

When opening an issue, include:

- WordPress version
- Elementor version
- Plugin version
- Theme name
- Steps to reproduce
- Expected result vs actual result

## Code of Conduct

Please be respectful and constructive in all discussions and reviews.
